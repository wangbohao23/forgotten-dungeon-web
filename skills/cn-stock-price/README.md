# 股票价格查询工具

一个简单的中国A股实时股票价格查询脚本。

## 功能

- 查询实时股票价格
- 支持单只或多只股票同时查询
- 提供默认股票列表（主要指数+个股）

## 安装

无需安装，确保已安装 Python 3 和 requests 库：

```bash
pip install requests
```

## 使用

### 基本用法

```bash
# 查询默认股票列表
python scripts/check.py

# 查询指定股票（支持多只）
python scripts/check.py sh600519
python scripts/check.py sh600036 sz002352
```

### 股票代码格式

- **上证股票**: `sh` + 6位数字，如 `sh600036`（招商银行）
- **深证股票**: `sz` + 6位数字，如 `sz002352`（顺丰控股）
- **指数基金**: 同上，如 `sh000300`（沪深300）

### 常用指数代码

| 代码     | 名称     |
| -------- | -------- |
| sh000985 | 中证全指 |
| sh000300 | 沪深300  |
| sh000905 | 中证500  |
| sh000852 | 中证1000 |
| sz399006 | 创业板指 |
| sh000001 | 上证指数 |
| sz399001 | 深证成指 |

### 输出示例

```
000985 中证全指 6156.42
000300 沪深300 4649.60
```

## 默认股票列表

当不指定参数时，查询以下股票：

1. 中证全指 (sh000985)
2. 沪深300 (sh000300)
3. 中证500 (sh000905)
4. 中证1000 (sh000852)
5. 创业板指 (sz399006)

## 数据来源

使用腾讯财经 API 获取实时数据。
